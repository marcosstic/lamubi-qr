# La Mubi - Landing Page Mockup

## Vista Previa Online

Para compartir este mockup con tu cliente:

### Opción 1: GitHub Pages (Recomendado)
1. Sube este repositorio a GitHub
2. Ve a Settings > Pages
3. Activa GitHub Pages desde la rama main
4. Obtendrás una URL pública como: `https://[tu-usuario].github.io/lamubi2/`

### Opción 2: Netlify Drop (Gratis)
1. Ve a https://app.netlify.com/drop
2. Arrastra la carpeta del proyecto
3. Obtén una URL pública instantánea

### Opción 3: Vercel (Gratis)
1. Ve a https://vercel.com
2. Importa tu repositorio de GitHub
3. Obtén una URL pública automáticamente
